# Локальный редактор когнитивного графа

Приложение хранит ориентированный граф в `networkx.DiGraph`, показывает его в локальном интерфейсе Streamlit и сохраняет данные в Excel.

## Запуск

Дважды щёлкните `run_graph_editor.bat`. При первом запуске скрипт создаст `.venv`, установит зависимости и откроет приложение в браузере.

Или вручную:

```powershell
py -m venv .venv
.venv\Scripts\python -m pip install -r requirements.txt
.venv\Scripts\python -m streamlit run app.py
```

Приложение слушает только `127.0.0.1`, поэтому другие устройства в общей Wi-Fi-сети не могут подключиться к нему. Не меняйте `server.address` на `0.0.0.0`, если не планируете отдельно настраивать аутентификацию и HTTPS. При первом запуске батник создаёт локальный ключ cookie в `.streamlit/cookie_secret.txt`; файл повторно используется при следующих запусках и не попадает в Git.

## Управление

- Вершина: введите название и нажмите «Добавить вершину»; существующую вершину можно выбрать и переименовать, при этом её связи сохранятся.
- Связь: выберите начало и конец, тип связи и вес от `-1` до `1`; для двусторонней связи толщина стрелок вперёд и обратно настраивается отдельно.
- При выборе уже связанной пары форма автоматически подставляет сохранённые веса и жирность обеих стрелок для редактирования.
- Цвет вершины выбирается в редакторе; входящие ненулевые стрелки автоматически получают цвет вершины назначения.
- Длинные названия переносятся внутри вершины и при необходимости сокращаются многоточием; полное название показывается при наведении.
- Двусторонняя связь создаёт два независимых ребра: `A → B` и `B → A`, каждому можно задать свой вес.
- Нулевой вес допустим и отображается серой линией без наконечника.
- Вершины и связи удаляются через списки в левой панели.
- Вершины можно перетаскивать мышью по расширенной рабочей плоскости за пределами исходного экрана; новые координаты остаются в текущем графе и записываются в Excel при сохранении.
- Короткий щелчок по вершине выбирает её в редакторе слева, а щелчок по стрелке подставляет обе вершины связи в форму редактирования.
- Вид графа можно плавно масштабировать от 10% до 300% колёсиком или жестом тачпада и перемещать перетаскиванием за любую область кроме вершин; кнопка «Весь граф» автоматически подбирает масштаб и центрирует все вершины, сетка покрывает всё поле и отключается отдельной кнопкой, а «На весь экран» разворачивает редактор на весь дисплей.
- «Перестроить граф» после подтверждения запускает `networkx.spring_layout`.
- При перестройке дополнительное раздвижение вершин сохраняет минимальное расстояние между ними, включая несвязанные вершины, и отодвигает вершины от проходящих мимо стрелок и подписей весов.
- Стрелки автоматически изгибаются вокруг посторонних вершин, а подписи весов размещаются без перекрытия вершин и других подписей.
- Парные стрелки двусторонней связи идут параллельно и прямо, если путь свободен; при препятствии они используют общую сторону изгиба. Подписи весов располагаются только с внешних сторон пары.
- Если другую подпись пересекает стрелка, подпись сдвигается от середины к ближайшему свободному месту вдоль своей связи.
- «Сохранить в папку проекта» записывает `graph_database.xlsx` рядом с приложением.
- Каждые 5 минут открытая сессия перезаписывает `graph_autosave.xlsx`; кнопка «Восстановить из автосохранения» загружает эту резервную копию.
- «Скачать Excel» отдаёт копию через браузер, а загрузчик принимает такую книгу обратно.

## Структура Excel

- `Вершины`: `id`, `label`, `x`, `y`, `color`.
- `Связи`: `source`, `target`, `weight`, `bold`.
- `Связи по названиям`: те же связи, но в `source` и `target` записаны названия вершин вместо их ID. Лист предназначен для просмотра; при загрузке граф восстанавливается по листу `Связи`.
- `Настройки`: версия формата и служебная информация.

Вес связи всегда должен быть числом в диапазоне `[-1; 1]`.
