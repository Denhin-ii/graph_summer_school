# Локальный редактор когнитивного графа

Приложение хранит ориентированный граф в `networkx.DiGraph`, показывает его в локальном интерфейсе Streamlit + Plotly и сохраняет данные в Excel.

## Запуск

Дважды щёлкните `run_graph_editor.bat`. При первом запуске скрипт создаст `.venv`, установит зависимости и откроет приложение в браузере.

Или вручную:

```powershell
py -m venv .venv
.venv\Scripts\python -m pip install -r requirements.txt
.venv\Scripts\python -m streamlit run app.py
```

## Управление

- Вершина: введите название и нажмите «Добавить вершину».
- Связь: выберите начало и конец, тип связи, вес от `-1` до `1` и при необходимости опцию «Жирная стрелка».
- Цвет вершины выбирается в редакторе; входящие ненулевые стрелки автоматически получают цвет вершины назначения.
- Двусторонняя связь создаёт два независимых ребра: `A → B` и `B → A`, каждому можно задать свой вес.
- Нулевой вес допустим и отображается серой линией без наконечника.
- Вершины и связи удаляются через списки в левой панели.
- «Перестроить граф» запускает `networkx.spring_layout`.
- «Сохранить в папку проекта» записывает `graph_database.xlsx` рядом с приложением.
- «Скачать Excel» отдаёт копию через браузер, а загрузчик принимает такую книгу обратно.

## Структура Excel

- `Вершины`: `id`, `label`, `x`, `y`, `color`.
- `Связи`: `source`, `target`, `weight`, `bold`.
- `Настройки`: версия формата и служебная информация.

Вес связи всегда должен быть числом в диапазоне `[-1; 1]`.
